League of Legends
==========

Each file can be open with notepad (right click > open with > notepad).

Description.
==========
Theme inspired by LoL ingame chat.

+ Each user got assigned a champion.
+ Subscriber got assigned to team color (blue)
+ Plebs got asssigned to enemy color (red)

- not bright screen friendly.
- timestamp, [all], hero making chat longer.
- sub only mode will obviously shows only blue color.
- please adjust text-shadow if its ugly/not visible.

Timestamp library Luxon.

if you find this thing confusing, don't be scared to dm me on twitter. I'll help you if i'm available or able to.

! Warning
==========
