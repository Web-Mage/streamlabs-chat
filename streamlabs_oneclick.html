Streamlabs Chat Box One Click Install

Last Update 24 April 2022

Installation Steps

  1. Click your chosen theme below.
  2. Then, on Streamlabs webpage, choose which 'Widget Theme' you want to install the chatbox into.
  3. Finally, scroll down and click on Custom Fields section and make changes as you please.
  4. You can then copy your widget url and use it on OBS.

Themes

  2. Pastel Heart Default
  3. Pastel Heart Colorful
  4. Pink With Gifs
  5. White Box
  6. User Color
